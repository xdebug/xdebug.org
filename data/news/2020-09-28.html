Xdebug 2.9.8 is out!

This patch release fixes a compile error on FreeBSD related to the TCP Keepalive Mechanism that was introduced in 2.9.7 and a path/branch coverage issue with loops.

The full list of changes can be found on the updates page.

The source code and Windows binaries can be found on the downloads page, and as usual, Xdebug is installable through PECL.
