Xdebug 3.3.2 is out!

This is a bug fix release that addresses a crash bug when Xdebug and Blackfire are active, as well as a crash bug/100% CPU bug in threaded PHP environments (Apache, FrankenPHP).

The full list of changes can be found on the updates page.

The source code can be found on the downloads page, and as usual, Xdebug is installable through PECL.

If you find a bug, please file a report at Xdebug's Issue Tracker.

This release also contains contributions by: Kévin Dunglas and Luc Vieillescazes — Thanks!
