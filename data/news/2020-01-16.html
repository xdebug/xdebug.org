Xdebug 2.9.1 is out!

This bug fix release restores step debugging performance as it was in Xdebug 2.7.2, while still maintaining the "resolved breakpoint" feature. Beyond improved performance for debugging, it also addresses a whole range of smaller issues.

The full list of changes can be found on the updates page.

The source code and Windows binaries can be found on the downloads page, and as usual, Xdebug is installable through PECL.
