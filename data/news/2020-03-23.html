Xdebug 2.9.4 is out!

This bug fix release addresses a crash bug introduced in Xdebug 2.9.3 that fixed an incompatibility with other extensions that overload opcodes. It also makes sure that the Step Debugger's 'detach' command does not disable debugging for the rest of the PHP process.

The full list of changes can be found on the updates page.

The source code and Windows binaries can be found on the downloads page, and as usual, Xdebug is installable through PECL.
