Xdebug 3.1.2 is out!

This is a bug fix release that fixes a few crash bugs related to PHP 8.1 fibers, a crash bug when Xdebug can't write a profiler file, and an issue with Xdebug's var_dump() not using the magic __debugInfo method.

The full list of changes can be found on the updates page.

The source code can be found on the downloads page, and as usual, Xdebug is installable through PECL.

If you find a bug, please file a report at Xdebug's Issue Tracker.
