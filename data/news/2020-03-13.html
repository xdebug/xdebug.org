Xdebug 2.9.3 is out!

This bug fix release addresses a bug with Xdebug's resolved breakpoints in inherited classes, a bug with outdated behaviour with regards to PHP's error_get_last(), and an incompatibility with other extensions that overload opcodes.

The full list of changes can be found on the updates page.

The source code and Windows binaries can be found on the downloads page, and as usual, Xdebug is installable through PECL.
