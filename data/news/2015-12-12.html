Xdebug 2.4.0RC3 is out!

This is the third release candidate of the 2.4 series. The 2.4 series adds support for PHP 7 and is in large parts a rewrite of the earlier code base to support PHP 5.4 through PHP 7.0. This release candidate fixes weird optimisation issues with GCC 4.8, and should make everything work will on Windows due to issue with changes in "long".

The full change log can be found on the updates page. The source code can be found on the downloads page, and as usual, Xdebug is installable through PECL.
