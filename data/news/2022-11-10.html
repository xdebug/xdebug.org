Xdebug 3.2.0RC2 is out!

This is a bug fix release that main fixes issues related to step debugging not always stepping through to the following line. It also fixes trace files with anonymous closures, and this release also incorporates the fixes from the 3.1.6 release.

The full list of changes can be found on the updates page.

The source code can be found on the downloads page, and as usual, Xdebug is installable through PECL.

If you find a bug, please file a report at Xdebug's Issue Tracker.
