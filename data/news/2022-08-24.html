Xdebug 3.2.0alpha3 is out!

This release forces the new return value debugging step and information to be on, to be able to test features in Xdebug that are not strictly supported yet in all IDEs. This is likely a temporary change, which has a chance of being reverted in the next release.

This is a source only release, without support for Windows.

The full list of changes can be found on the updates page.

The source code can be found on the downloads page, and as usual, Xdebug is installable through PECL.

If you find a bug, please file a report at Xdebug's Issue Tracker.
