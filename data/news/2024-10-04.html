Xdebug 3.4.0beta1 is out!

This is a beta release to allow for testing Xdebug 3.4 with PHP 8.4. There will be further fixes and other changes.

The full list of changes can be found on the updates page.

The source code can be found on the downloads page, and as usual, Xdebug is installable through PECL.

If you find a bug, please file a report at Xdebug's Issue Tracker.

This release also contains contributions by: Christoph Becker and Tim Düsterhus — Thanks!
