Xdebug 2.9.5 is out!

This bug fix release addresses two crash bugs; one when an exception was thrown in a destructor, and another one where another extension compiles a PHP file during request start-up. A Code Coverage anomaly is also resolved.

The full list of changes can be found on the updates page.

The source code and Windows binaries can be found on the downloads page, and as usual, Xdebug is installable through PECL.
