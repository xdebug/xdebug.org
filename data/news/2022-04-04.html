Xdebug 3.1.4 is out!

This is a bug fix release that addresses bugs related to the step debugger, including debugging ArrayObject objects, crashes with closures with unset static properties, and a crash when removing a duplicate "call" or "return" breakpoint. It also fixes missing static array assignment lines with code coverage, and adds a warning when zlib compression with xdebug.profiler_append is used.

The full list of changes can be found on the updates page.

The source code can be found on the downloads page, and as usual, Xdebug is installable through PECL.

If you find a bug, please file a report at Xdebug's Issue Tracker.
