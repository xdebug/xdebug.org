Xdebug 2.5.4 is out!

This release fixes an infinite loop with Xdebug 2.5.3 if the pcntl extension is not statically loaded and set_time_limit is used, as well as crash on 32 bit platforms, and another issue with code coverage.

The full change log can be found on the updates page. The source code can be found on the downloads page, and as usual, Xdebug is installable through PECL.
