Xdebug 2.8.1 is out!

This is the first bug fix release of the Xdebug 2.8 series.

This bug fix release only addresses a slow down with code coverage that was introduced in Xdebug 2.8.0.

The full list of changes can be found on the updates page.

The source code and Windows binaries can be found on the downloads page, and as usual, Xdebug is installable through PECL.
