Xdebug 2.4.0RC1 is out!

This is the first release candidate of the 2.4 series. The 2.4 series adds support for PHP 7 and is in large parts a rewrite of the earlier code base to support PHP 5.4 through PHP 7.0. This release candidate fixes a few bugs found in the 2.4.0beta1 release. Among others problems with code coverage, tracing and remote debugging. It also adds support for Windows.

The full change log can be found on the updates page. The source code can be found on the downloads page, and as usual, Xdebug is installable through PECL.
