Xdebug 2.6.1 released!

This is the first bug fix release of the 2.6 series.

This release fixes several crash bugs, a memory leak, and an issue with step debugging where object properties with numerical names could not be inspected.

The full list of changes can be found on the updates page.

The source code and Windows binaries can be found on the downloads page, and as usual, Xdebug is installable through PECL.
